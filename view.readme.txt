Plugin for Todo.txt CLI

Plugin Name : view
Author      : Mark Wu http://blog.markplace.net
License     : GPL,  http://www.gnu.org/copyleft/gpl.html
Inspired    : By Paul Mansfield's projectview from http://github.com/the1ts/todo.txt-plugins/blob/master/README.projects
Description : Show todo items containing TERM, grouped by [OPTION], and displayed in priority order. If no TERM provided, displays entire todo.txt.
Usage       :
  todo.sh view [OPTION] [TERM]
  [OPTION]
     project : Summarized by project ( +project )
     contact : Summarized by contact ( @contact )
     date    : Summarized by date ( t:date )
Example     :
  $ todo.sh view project          # Show todo items by project
  $ todo.sh view project @contact # Show todo items by project and filter by @contact
  $ todo.sh view date +project    # Show todo items by date and filter by +project
